% existing spine-care disparities. EU AI Act -> bias audits are a deployment
% precondition, not optional. Keep tight (3-4 sentences).

% --- Para 2: Why segmentation fairness is hard (sets up the ruler) ------------
% (i) output is millions of voxels, not one decision -> must reduce to a scalar
%     (Dice/HD95), and that reduction is itself volume-biased when anatomy differs
%     by group. (ii) annotations are expensive + subjective -> datasets MIX expert
%     (gold) and machine (silver) labels. If silver carries bias, the *reference*
%     used to judge fairness is unreliable.
